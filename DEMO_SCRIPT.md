### Що показуємо:
Як швидко додати нову витрату

### Кроки:
1. **Натискаємо "Додати транзакцію"**
   ```
   "Додамо нову витрату"
   ```

2. **Заповнюємо форму:**
   - Тип: Витрата (обрано за замовчуванням)
   - Сума: "120"
   - Категорія: "Їжа" (з випадаючого списку)
   - Опис: "Обід в кафе"
   - Дата: (сьогодні)

3. **Натискаємо Зберегти**
   ```
   "Зберігаємо"
   ```

4. **Показуємо оновлення**
   ```
   "І одразу бачимо оновлення - баланс зменшився, транзакція з'явилась"
   ```

### Що говорити:
> "Всі операції миттєві, без перезавантаження сторінки. Категорії можна налаштовувати під себе."
